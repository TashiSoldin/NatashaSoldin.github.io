Electrical Engineer Project Link

MAPWAPS

I am currently completing my final year thesis entitles MAPWAPS which stands for the mapping of woody alien plantation. This project uses satellite imagery to track the occurence and spatial distribution of alien invasive plantation (IAP). IAPs can cause serious ecosystem decline and these maps will aid rehabilitation programs eradicate them efficiently.

Engineering Design Github Repository Report

Automated Bird Scale

During a fourth year engineering design course, my group and I built an automated bird scale to assist the ornithology department in their monitoring of forked-tail drongos weight patterns. This system was equipped with a load cell sensor connected to a microcontroller which implements a data processing algorithm to obtain an accurate weight value.

Digital Signals Processing Github Repository Report

Electrical Component Classifier

This project focuses on the classification of electrical components using a machine learning algorithm. This is achieved with a Convolutional Neural Network coupled with transfer learning. This confusion matrix shows that the classification algorithm executed with great accuracy.

Embedded Systems Github Repository Report

VADER

This project implements a hashed password recovery system. It utilises a Field Programmable Gate Array (FPGA) that uses parallel programming techniques to complete computationally expensive tasks. It begins with a dictionary attack followed by a brute force attack to ensure password recovery.

Control Systems Report

Helicopter Controller

This control practical included the system plant identification of a provided MBUct helicopter simulation system through step and frequency response methods, derivation of a transfer function, design of suitable controller through root locus and frequency design methods and controller implementation via analogue circuit.

Engineering Design Github Repository Report

IMU Data Compression and Encryption

During a third year engineering design course, my partner and I built an ARM based digital IP using an STM32F051- microcontroller that retrieves, compresses and encrypts data from an Inertial Measurement Unit (IMU) sensor. This was designed to be implemented as a buoy installed on an ice ’pancake’ in the Southern Ocean to collect data about the ice and wave dynamics.
